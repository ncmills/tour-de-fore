01 Brand essence

Raunchy bachelor golf-trip with a dive-bar-after-dark soul. The North Star is "dark but readable" — near-black zinc surfaces, one hot ember accent, type that swings from clean to neon-sign depending on the scene.

🍺 neon bar sign at 1am· ⛳ vintage golf crest gone rogue· 🎰 Vegas marquee

02 Logo & brand mark

08 Accessibility & print

  • ▸ Focus ring: 2px ember outline + 4px glow on :focus-visible.
  • ▸ Skip link: .skip-to-content on every page → #main-content.
  • ▸ Tap targets: 44×44px minimum on mobile, preserved even when visuals shrink.
  • ▸ Responsive: all overrides scoped to @media (max-width:640px) — mobile fixes never break desktop.
  • ▸ Print: full @media print stylesheet — itineraries go light, nav/animations stripped, URLs appended.

09 Do / Don't

Do
Don't
  • ✓ Ember #EA580C as the single accent
  • ✓ Reuse a primitive before inline-styling
  • ✓ Pick fonts by tier
  • ✓ Radius/spacing from the scale (buttons = 10px)
  • ✓ Preserve 44px tap targets
  • ✓ Respect prefers-reduced-motion
  • ✗ A second brand/accent color (no blue/purple)
  • ✗ text-transform:uppercase on headings
  • ✗ Inline-style a one-off when a primitive exists
  • ✗ Promote a Tier-3 scene font to a global heading
  • ✗ Invent a radius value off the scale
  • ✗ Let a mobile fix change desktop · go corporate
